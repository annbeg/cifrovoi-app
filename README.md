# Запуск приложения

Для запуска приложения понадобится пакетный менеджер **npm**.

Перед стартом перейдите в директорию приложения и выполните команду

### `npm install`

для установки необходимых зависимостей.

Теперь запустите приложение командой 

### `npm start`
